首先要看下链表的翻转方法

```java
    public ListNode reverseList(ListNode head) {
        //判空
        if(head == null){
            return null;
        }
        //记录反转前的最后节点
        ListNode index = null;
        while(head != null){
            //找到下一个节点
            ListNode next = head.next;
            //断开链接，链接到上一个头结点
            head.next = index;
            //记录翻转前头节点
            index = head;
            //头节点下移
            head = next;
        }
        return index;
    }
```
